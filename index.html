ABOUT DUCKYTV - BETA V1.3.0

DuckyTV is a Webapp to stream Movies/TV Shows/(Web)Radio.
To achieve this, it uses a so called 'scraper' and different APIs which get most of the desired content.
Streaming is a grey area, the more you look into it, the more complicated it gets.

The Core of DuckyTV is 100% Open-Source and can be viewed @ github.com/duckforceone/IcVR
USED LIBRARIES:
  • jQuery
  • Bootstrap
  • TMDB API
  • videospider.in API
  • radio-browser.info API
  • My own JSON Compilation of German TV Stations
  • hls.js for the .m3u decompilation
